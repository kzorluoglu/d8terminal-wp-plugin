== Description ==
D8 Linux Terminal Widget is a unique terminal widget for your WordPress site. It allows users to interact with a simulated Linux terminal environment directly from your website.

== Installation ==
1. Upload `d8-linux-terminal-widget` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the widget by adding it to your sidebar through the 'Appearance > Widgets' menu in WordPress.
